Highlight·Scout

Search every reading highlight you've ever saved — in one place, on your machine.

Download View source

Free & open source (MIT) · macOS & Windows · your data never leaves your computer.

Import from anywhere

CSV

Any export. A mapping screen lines your columns up with title / author / text / note; it remembers the mapping.

Kindle

Your device's My Clippings.txt — highlights and notes.

JSON

Our own format (and what Export writes) — script an importer for any tool you like.

Readwise & Zotero

Optional connectors. Zotero is read locally — colours and types kept; no account needed.

Re-importing the same file never makes duplicates.

Then find anything, fast

Instant keyword search

Sub-second over tens of thousands of highlights, with a real query language: "phrases", -exclude, prefix*, au: tag: y:2023, regex.

Read in context

A reading pane with matched terms lit up, citations, collections, and inline images.

Sort, group, filter

By work, author, year, tag, source, or annotation colour.

Semantic & related

Optional meaning-based search and “find related”, powered by a local engine (QMD) if you install it.

Get started

macOS: open the .dmg, drag to Applications, then right-click → Open the first time.
Windows: unzip the portable build and run highlight-scout.exe — no installer.

Press ⌘⌥⇧H to summon it from anywhere, choose Import ▾, and search.
