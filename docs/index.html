Get started

macOS: open the .dmg and drag Highlight Scout to Applications. The app isn't notarised yet, so the first launch is blocked — open System Settings → Privacy & Security and click Open Anyway. (Or in Terminal: xattr -dr com.apple.quarantine "/Applications/Highlight Scout.app".)
Windows: unzip the portable build and run highlight-scout.exe — no installer; at the SmartScreen prompt choose More info → Run anyway.

Heads up: the downloads aren't code-signed/notarised yet, so both systems will warn you the first time (that's the steps above, not a problem with the app). They're also built automatically and not yet tested on a clean machine — if anything misbehaves, please open an issue on GitHub.
